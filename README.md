📊 Business Performance Analysis Dashboard
Overview

Built an interactive Excel dashboard to analyze sales, profitability, and discount impact using the Superstore dataset.

Key Features

KPI metrics (Sales, Profit, Orders, Profit Margin)

Region-wise and Segment-wise analysis

Product performance evaluation

Discount vs Profit relationship analysis

Interactive slicers for dynamic filtering

Tools Used

Microsoft Excel

Pivot Tables

Data Visualization

Business Analysis
